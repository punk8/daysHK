# lib/shared/theme

## 职责

定义 App 设计系统。

## 视觉方向

- 浅色背景。
- 白色卡片。
- 深墨色正文。
- 克制香港红作为提醒 / 离港强调色。
- logo 蓝作为主要操作和在港状态色。
- 卡片圆角约 8px。
- 信息密度适中，适合长期反复查看。

## 建议 token

```pseudo
colors:
  background = #F5F8FF
  surface = #FFFFFF
  textPrimary = #17233A
  textSecondary = #6D7890
  brandBlue = #326BFF
  hkRed = #D92D3A
  warningBg = #FFF7E8
  dangerBg = #FFF1F1

radii:
  card = 8
  button = 8
```
